# Describe
This build a Rails environment with Docker.  

# Version Information

| item         | version    |
|:-------------|:-----------|
| Docker Image | ruby:2.6.3 |
| Rails        | 5.2.1      |

# How To Use

Execute `setup.sh`. Then Rails app is generated in `myapp` directory.  

If you want to initialize, run `setup.sh` again.
